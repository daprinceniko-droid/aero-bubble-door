# What this is
A personal portfolio built to communicate both professional competence and creative identity to recruiters. Each section has its own distinct visual world, drawing on personal influences from Y2K internet culture, early 2000s gaming, and underground car culture.

# Context
This is Version 2 of the portfolio. Version 1 was a gamified Windows XP desktop environment with interconnected minigames that unlocked the portfolio content — a Club Penguin battle, an aquarium app, and a drag race. It was scrapped after user testing revealed the puzzle mechanic was inaccessible to anyone without guidance. V1 lives at penguin-canvas-wander.lovable.app.
V2 keeps the personality but puts the content first. The gamification exists as a layer on top of a straightforward linear structure rather than as the structure itself.
Full design process documented in the project case study.
